Vacation-browser

vacation browser = wget + crontab + filesave/manager. Let's you save screenshots of webpages when you are on vacation so that you don't miss front page stories when you come back

View the Project on GitHub srinikom/vacation-browser

git clone git://github.com/srinikom/vacation-browser.git
cd vbr; ls

Follow instructions from here. http://www.fabulasolutions.com/2010/02/dont-miss-news-and-trends-when-you-are.html
